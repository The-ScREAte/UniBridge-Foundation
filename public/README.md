# IMPORTANT: Add Your Logo

Please save the UniBridge logo image as `logo.png` in this folder.

The logo image you provided should be saved here for the website to display it correctly.

**File requirements:**
- Filename: `logo.png`
- Format: PNG (recommended for transparency)
- Recommended size: 200x200px or larger (maintains aspect ratio)
- The logo will be automatically resized by the website

After adding the logo, the website will display it in:
- Navigation bar
- Hero section
- Admin login page
- Footer
